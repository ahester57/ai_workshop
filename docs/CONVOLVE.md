# Convolve

## Usage

```bash
$ simulated_annealing convolve -h
usage: simulated_annealing convolve [-h] -f IMAGE_FILENAME [-i ITERATIONS]

options:
  -h, --help            show this help message and exit
  -f IMAGE_FILENAME, --file IMAGE_FILENAME
                        .csv file containing your image as text
  -i ITERATIONS, --iterations ITERATIONS
                        Number of iterations to process. Default 2.
(.venv-ai)
```
